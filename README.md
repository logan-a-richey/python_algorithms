# Python Algorithms

This repository contains a collection of 67 different algorithms implemented in Python. The purpose of this project is to explore and learn how to implement fundamental algorithms from various domains, including sorting, searching, graph theory, dynamic programming, and more.

Each algorithm is grouped into categories, with each category contained in its own class. Running the classâ€™s main function will execute all algorithms within that category.

## Categories

- **Sorting Algorithms**
- **Searching Algorithms**
- **Graph Algorithms**
- **Dynamic Programming (DP)**
- **String Algorithms**
- **Data Structures**
- **Mathematical Algorithms**
- **Greedy Algorithms**
- **Backtracking Algorithms**
- **Divide and Conquer Algorithms**
